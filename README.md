# MyIdentity Portfolio

A modern, high-performance, and visually striking personal portfolio built for **Shaker Al-Muzayni**, a Senior Laravel Developer & DevOps Engineer. The project is designed with a dark, neural "Obsidian" aesthetic, utilizing glassmorphism and subtle neon accents to convey technical proficiency and modern design standards.

## ✨ Key Features

- **Premium Dark Aesthetic**: A carefully crafted "Obsidian & Deep Violet" color palette that looks professional and modern.
- **Dynamic Routing**: Built with `react-router-dom` to support a clean single-page experience while allowing deep-linking to specific project case studies.
- **Interactive UI**: Powered by `framer-motion` for smooth scroll animations, micro-interactions, and page transitions.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing with an interactive mobile hamburger menu.
- **Glassmorphism Design**: Extensive use of background blurs, semi-transparent borders, and layered depths.
- **Arabic Typography Support**: Integrated with the Google "Cairo" font for excellent readability in both English and Arabic.

## 🛠️ Tech Stack

- **React 19** (Vite)
- **Tailwind CSS 4** (Utility-first styling & custom themes)
- **Framer Motion** (Complex animations & gestures)
- **React Router DOM** (Client-side routing)
- **Lucide React** (Clean, modern SVG icons)

## 📂 Project Structure

```text
src/
├── assets/         # Images, icons, and static assets

├── index.css       # Global styles, Tailwind theme config, and custom animations
```

## 💻 Local Development

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shaker-Al-Muzaini/protofilemy.git
   cd protofilemy
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
